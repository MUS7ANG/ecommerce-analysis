# E-Commerce Sales Analysis

Анализ данных e-commerce магазина: 536,350 транзакций за 1 год, 4,739 клиентов, 3,768 товаров, 38 стран. Общая выручка: 60,283,541.

## Стек

- Python 3 (pandas, seaborn, matplotlib)
- Jupyter Notebook
- SQL для дополнительного анализа
- PostgreSQL

---

## Исследуемые вопросы и выводы

### 1. Какие товары наиболее прибыльны?

Топ-3 товара (Popcorn Holder, World War 2 Gliders, Paper Craft Little Birdie) дают 2.7% выручки. Для 80% дохода нужно 945 товаров из 3,768 (25% каталога). Топ-50% товаров дают 95.5% выручки.

Вывод: 1,985 товаров (53%) приносят менее 0.01% выручки и подлежат удалению.

### 2. Какие клиенты приносят основную выручку?

Топ-5 клиентов дают 9.5% выручки. Лучший клиент принес 2.1M за 74 заказа. 70% клиентов повторяют покупки.

Вывод: Высокая концентрация в топ-клиентах, хорошая лояльность. Нужны персональные менеджеры для топ-10.

### 3. Есть ли сезонность?

Колебания выручки: минимум 22,892, максимум 851,652 (409.7% разброс). Пики вероятно в ноябре-декабре.

Вывод: Высокая сезонность требует гибкой системы управления запасами и логистики.

### 4. Какие страны наиболее перспективны?

UK: 49.9M (82.9% выручки). Топ-3 страны дают 89.3% дохода. Потенциальные рынки: Germany (92 клиента), France (87 клиентов), Netherlands (средний чек 22,142).

Вывод: Критическая концентрация в UK. Нужна диверсификация через маркетинг в Germany, France, Australia.

### 5. Стоит ли расширять ассортимент?

HHI индекс = 15 (низкая концентрация). 53% товаров приносят менее 0.01% выручки.

Вывод: Расширение не критично. Приоритет: удалить мертвый товар, оптимизировать топ-50 товаров.
